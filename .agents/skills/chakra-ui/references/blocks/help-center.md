# Help Center

Components for building a help and support center.

> **Note:** Blocks require `CHAKRA_PRO_API_KEY` environment variable. If not set, block features are unavailable.

## Group

marketing

## When to Use

- ヘルプセンター・サポートページを構築するとき
- カテゴリ別のヘルプ記事コレクションを表示するとき
- ヘルプセクションのカード・リスト表示が必要なとき

## MCP Parameters

Tool: `get_component_templates`

| Parameter | Type | Description |
|-----------|------|-------------|
| `category` | `string` | 固定値: `"help-center"` |
| `id` | `string` | 取得するバリアントの ID |
